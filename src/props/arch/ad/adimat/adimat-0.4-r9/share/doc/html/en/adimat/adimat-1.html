1. Introduction

ADiMat is a tool to differentiate Matlab code using a source transformation approach implementing the forward mode of automatic differentiation. All commonly used language features of Matlab 6 and 6.5 are supported, for a detailed description look at chapter Supported Matlab constructs. ADiMat's development is in progress. New features will be added in the future.
