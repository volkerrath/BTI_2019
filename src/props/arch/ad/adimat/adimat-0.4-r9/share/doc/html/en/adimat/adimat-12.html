12. Questions and Answers

What is the order of derivative objects and results or parameters in a function call?

The order of derivate objects and results or parameters in a function is: The derivative object preceeds its corresponding active result or active parameter. For example: Let [y,w]=f(a,b,c) be a function having the parameters a,b, and c and the results y and w. Assume that the parameters a and c and the output y are active. The correct call of this function is: [g_y,y,w]=f(g_a,a,b,g_c,c).

Is the derivative expression put in front of the orginial expression?

The derivative expression is put in front of the original expression usually. The only exception to this rule is the backslash operator, where the original expression is assigned to a temporary variable, the derivative expression is executed, which uses the temporary variable, and after that the temporary expression is assigned to its orinigal variable.

Which prefix do derivative objects get?

Derivative objects are prefixed using 'g_' by default. This may be changed using the command line option --gradprefix=''.

Which prefix do differentiated functions get?

A differentiated function gets the prefix 'g_' by default. This may be changed using the command line option --funcprefix=''.
